JavaScript Basics

Variablen

JavaScript verwendet dynamische Typisierung (keine feste Typbindung). Variablen werden mit let definiert.

Gefahr

var nicht verwenden

  • let gilt nur im Gültigkeitsbereich (Block in dem es deklariert wurde, wie bei Java)
  • var gilt überall, egal wo und wann die Variable deklariert wurde
  • var erlaubt es die gleiche Variable mehrfach zu deklarieren
  • var führt schnell zu Fehlern oder unerwartetem Verhalten

Strings

Strings sind Zeichenketten, die Text darstellen und in Anführungszeichen (', oder ") geschrieben werden

Tipp

Strings lassen sich auch mit Template-Literalen (`Text ${variable}`) dynamisch zusammenstellen, was die Arbeit mit Variablen erleichtert. Variablen werden über ${variableName} direkt in den Text integriert, welcher dazu von Forwardticks ` umschlossen sein muss.

foreach-Schleife

  • iteriert über alle Werte in einer Liste von Werten (Array, String, …)
  • mit in wird auf den Index zugegriffen
  • mit of wird auf den Wert zugegriffen
