HTML Grundbegriffe

HTML

  • Webseiten sind in HTML (Hypertext Markup Language) geschrieben.
  • Browser verwenden eine Rendering-Engine, um HTML-Seiten korrekt anzuzeigen.

Browser Developer Tools

  • Ein Toolset im Browser zur Analyse und Fehlerbehebung von Webseiten.
  • Anleitung zum Öffnen der Dev Tools in verschiedenen Browsern
  • Mit einem Rechtsklick und der Auswahl von “Untersuchen” kann ein bestimmtes Element auf einer Webseite inspiziert werden.

HTML Tags

  • “Tagging” bedeutet, Elemente zu markieren oder zu kennzeichnen. (“to tag” = markieren, auszeichnen)
  • Tags werden durch spitze Klammern (< >) eingeschlossen, z. B. <html lang="de">, </head>, <br/>
  • Ein öffnendes Tag sieht so aus: <html lang="de">, <head>, <body>, usw.
  • Ein schließendes Tag enthält einen Schrägstrich /: </html>, </head>, </body>, usw.
  • Ein Element besteht aus einem öffnenden Tag, dem Inhalt und einem schließenden Tag.
  • Elemente ohne Inhalt (leere Tags): z. B. <br />, <br>, <meta charset="utf-8" /> 1
  • Weitere Informationen auf der W3C Webseite

HTML Attribute

  • Attribute enthalten zusätzliche Informationen zu einem HTML-Element.
  • Sie werden im öffnenden Tag hinter dem Tagnamen angegeben, z. B. <html lang="de">, <meta charset="utf-8" />
  • Syntax: Attributname = “Wert” 2
  • Mehrere Attribute werden durch Leerzeichen getrennt, z. B. <html lang="de" class="xyz">
  • Boolesche Attribute haben keinen Wert, da ihre bloße Anwesenheit „true“ bedeutet, z. B. <p hidden>

HTML Baumstruktur

  • HTML-Dokumente haben eine hierarchische Baumstruktur, bei der Elemente ineinander verschachtelt sind.

Exkurs: Baumstruktur

BegriffBedeutung
BaumEine hierarchische Datenstruktur in der Informatik
KnotenDatenelemente im Baum
KantenVerbindung zwischen den Datenelementen
EbenenHierarchieebenen von null bis beliebig viele
ElternelementEin Knoten, der mit darunterliegenden Elementen verbunden ist
KindelementEin Knoten, der mit darüberliegenden Elementen verbunden ist
Innerer KnotenEin Knoten, der sowohl Eltern- als auch Kindelemente hat

HTML = Hypertext Markup Language

  • Auszeichnungssprache (Markup-Sprache) zur Beschreibung der Struktur von Webseiteninhalten
  • Design und Layout werden durch CSS (Cascading Style Sheets) separat gestaltet
  • Keine Programmiersprache, sondern eine Markup-Sprache zur Strukturierung von Dokumenten
  • Aktuelle Version: HTML5 (eingeführt 2014)
