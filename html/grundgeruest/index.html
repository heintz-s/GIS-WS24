HTML Grundgerüst

HTML-Seite

Beispielcode für eine einfache HTML-Seite:

  • Verwende einen beliebigen Text-Editor (z. B. Visual Studio Code).
  • Kopiere den obigen HTML-Code und speichere ihn als Datei mit der Endung “.html” (z. B. index.html für die Startseite).
  • Öffne die Datei im Browser.
  • Der HTML-Renderer (Browser-Engine) interpretiert den HTML-Code und stellt die Seite dar.

Webseite mit mehreren Unterseiten

  • Eine Webseite besteht meist aus mehreren HTML-Dateien, die über Hyperlinks (z. B. in einem Navigationsmenü) miteinander verbunden sind.
  • Beispiel: index.html als Startseite, about.html, contact.html, gallery.html usw.
  • Diese Dateien können sich sich alle im Hauptverzeichnis befinden (z. B. https://webseite.de/index.html und https://webseite.de/about.html).
  • Alternativ können die Unterseiten in einem Unterordner liegen (z. B. im Uterordner “page” -> https://webseite.de/page/about.html).
  • Gleichbleibende Elemente (z. B. header, footer, navigation) wiederholen sich auf den Seiten, können aber zur Optimierung auch durch Code eingebunden werden.
