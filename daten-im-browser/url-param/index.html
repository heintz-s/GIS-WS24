URL Parameter

URL-Parameter ermöglichen die Übergabe kleiner Informationsinhalte über die URL.

  • Sie werden an die URL nach einem Fragezeichen (?) angehängt
  • und in der Form parameterName=parameterWert geschrieben.
  • Mehrere Parameter können mit einem & verknüpft werden.
  • Es können nur die für eine URL zulässigen Zeichen verwendet werden.

Beispiel-URL:

https://www.hs-furtwangen.de/suche?tx_solr%5Bq%5D=studieninfo

Parameter in URLs sollten den Seiteninhalt spezifizieren und inhaltlich klar definieren, z. B. durch eine eindeutige ID für ein angezeigtes Element.

Alternativ direkt in URL einbauen

Alternativ können Informationen auch in den URL Pfad mit eingebaut werden, z. B. kann die ID direkt Teil des Pfads sein (/products/123). Die Informationen können dann etwas schwerer zu extrahieren sein. In vielen Frameworks ist diese Methode hingegen üblich und gut unterstützt.
